45-Day DSA Learning Plan

June 9: Introduction to DSA, importance, and complexity analysis (Big O, Big Ω, Big Θ).
June 10: Arrays - Introduction, operations, and implementation.
  • Write a program to find the second largest element in an array.
  • Implement a function to rotate an array by `k` positions.
June 11: Strings - Basic operations, functions, and use cases.
  • Write a function to check if a string is a palindrome.
  • Implement a function to find the longest common prefix among a list of strings.
June 12: Linked List - Introduction, singly linked list, operations (insert, delete, traverse).
  • Implement a function to reverse a singly linked list.
  • Write a function to detect and remove a cycle in a linked list.
June 13: Doubly Linked List - Implementation and operations.
  • Implement insertion and deletion operations for a doubly linked list.
June 14: Circular Linked List - Implementation and operations.
  • Write a program to split a circular linked list into two halves.
June 15: Review and coding practice - Arrays and Linked Lists.
June 16: Stacks - Introduction, implementation using arrays and linked lists.
  • Implement a stack using two queues.
June 17: Stacks - Applications (infix to postfix, postfix evaluation).
  • Write a program to evaluate a postfix expression.
June 18: Queues - Introduction, implementation using arrays and linked lists.
  • Implement a queue using two stacks.
June 19: Circular Queue - Implementation and operations.
  • Write a program to generate binary numbers from 1 to `n` using a queue.
June 20: Priority Queue - Introduction and heap implementation.
  • Implement a priority queue using a heap.
June 21: Deque (Double-ended queue) - Implementation and operations.
  • Write a program to implement a deque and perform basic operations like insert, delete, and access elements.
June 22: Review and coding practice - Stacks and Queues.
June 23: Introduction to Trees - Terminologies, types, and basic concepts.
June 24: Binary Trees - Implementation and traversal techniques (in-order, pre-order, post-order).
  • Write a program to find the height of a binary tree.
  • Implement a function to print the level order traversal of a binary tree.
June 25: Binary Search Tree (BST) - Implementation, insertion, deletion, and traversal.
  • Write a function to find the lowest common ancestor (LCA) of two nodes in a BST.
  • Implement a function to convert a binary tree to a doubly linked list.
June 26: AVL Trees - Introduction, rotations, and balancing.
  • Write a program to insert elements into an AVL tree and ensure it remains balanced.
June 27: Heap Trees - Min-Heap and Max-Heap, implementation.
  • Implement a function to build a heap from an array.
  • Write a function to perform heap sort on an array.
June 28: Trie - Introduction and implementation.
  • Implement a trie and functions to insert and search for words.
June 29: Review and coding practice - Trees.
June 30: Graphs - Introduction, representations (adjacency matrix, adjacency list).
  • Implement a graph using an adjacency list and an adjacency matrix.
July 1: Depth First Search (DFS) - Theory and implementation.
  • Write a program to perform DFS on a graph and detect cycles.
July 2: Breadth First Search (BFS) - Theory and implementation.
  • Implement BFS to find the shortest path in an unweighted graph.
July 3: Shortest Path Algorithms - Dijkstra's algorithm.
  • Implement Dijkstra's algorithm to find the shortest path in a weighted graph.
July 4: Shortest Path Algorithms - Bellman-Ford algorithm.
  • Implement the Bellman-Ford algorithm to find the shortest path in a graph that may have negative weight edges.
July 5: Minimum Spanning Tree - Prim's and Kruskal's algorithms.
  • Write a program to find the Minimum Spanning Tree using Prim's algorithm.
  • Write a program to find the Minimum Spanning Tree using Kruskal's algorithm.
July 6: Graphs - Topological sorting and applications.
  • Implement a function to perform topological sort on a directed acyclic graph.
July 7: Review and coding practice - Graphs.
July 8: Hashing - Introduction, hash functions, and collision resolution techniques.
  • Implement a hash table with separate chaining for collision resolution.
July 9: Hashing - Applications and problems.
  • Write a function to check if two strings are anagrams using hashing.
  • Implement a function to find the first non-repeating character in a string using hashing.
July 10: Recursion - Introduction and basic problems.
  • Write a program to solve the Tower of Hanoi problem.
  • Implement a function to generate all permutations of a string using recursion.
July 11: Dynamic Programming - Introduction, memoization, and tabulation.
  • Implement the Fibonacci sequence using dynamic programming.
  • Write a program to solve the knapsack problem using dynamic programming.
July 12: Dynamic Programming - Advanced problems and techniques.
  • Implement a function to find the longest increasing subsequence in an array using dynamic programming.
  • Write a program to solve the edit distance problem using dynamic programming.
July 13: Greedy Algorithms - Introduction and basic problems.
  • Write a program to find the minimum number of coins needed to make a given amount using a greedy approach.
  • Implement a function to find the maximum number of activities that can be performed by a single person given a set of activities with start and end times.
July 14: Review and coding practice - Advanced topics.
July 15: Sorting Algorithms - Quick sort, merge sort.
  • Implement the quick sort algorithm.
  • Implement the merge sort algorithm.
July 16: Sorting Algorithms - Counting sort, radix sort.
  • Implement the counting sort algorithm.
  • Implement the radix sort algorithm.
July 17: Searching Algorithms - Binary search, linear search.
  • Write a program to implement binary search.
  • Implement a function to perform linear search in an array.
July 18: Backtracking - Introduction and basic problems.
  • Write a program to solve the N-Queens problem using backtracking.
  • Implement a function to solve the Sudoku puzzle using backtracking.
July 19: Advanced Backtracking - Problems and techniques.
  • Implement a function to generate all subsets of a set using backtracking.
  • Write a program to solve the Hamiltonian Path problem using backtracking.
July 20: Divide and Conquer - Introduction and problems.
  • Implement a function to find the closest pair of points in a set using the divide and conquer approach.
  • Write a program to solve the maximum subarray sum problem using divide and conquer.
July 21: Review and coding practice - Algorithms.
July 22: Mock Interviews - Conduct mock interviews with peers or use online platforms.
July 23: Final review, solving complex problems, and preparation for interviews.
